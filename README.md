# 📸 Image Gallery

A responsive and interactive Image Gallery built using HTML, CSS, and JavaScript. This project allows users to view images in a clean gallery layout and open them in a lightbox for a better viewing experience.

## 🚀 Features

- Responsive gallery layout
- Lightbox image preview
- Next & Previous navigation
- Close button
- Smooth hover effects
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
